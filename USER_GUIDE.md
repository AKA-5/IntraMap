## Getting Started

### What is IntraMap?

IntraMap is an interactive floor plan creator and viewer. Perfect for:
- Shopping malls
- Office buildings
- Universities
- Hospitals
- Event venues
- Any indoor space that needs navigation

### Quick Start

1. **Visit** the IntraMap homepage
2. **Click** "Create New Map" to start editing
3. **Add shapes** and icons to build your floor plan
4. **Save** to the cloud and get a shareable QR code
5. **Share** the QR code so visitors can navigate your space

## Troubleshooting

### Common Issues & Solutions

#### "Can't pan/drag the map"
**Fixed!** This was a bug in v1.0. Make sure you have the latest version (v2.0+).

**Desktop:**
- Click and drag on empty space (not on objects)
- Or hold Space key and drag anywhere

**Mobile:**
- Touch and drag with one finger
- Make sure you're not accidentally zooming (two fingers)

#### "Zoom buttons not showing"
The zoom buttons are always visible in the bottom-right corner.

If you don't see them:
- Scroll down‚Äîthey might be below fold on small screens
- Check your browser zoom (should be 100%)
- Try refreshing the page

#### "My changes aren't saving"
Make sure to:
1. Click **‚òÅÔ∏è Save to Cloud** (not just draft save)
2. Wait for confirmation message
3. Save your building ID/link
4. Draft saves are local only (browser storage)

#### "Search isn't finding my locations"
Check if:
1. Object has a Label in properties panel
2. Object has Tags for alternate keywords
3. You're searching the right floor
4. Spelling matches label or tags

#### "Objects are locked/can't select"
In Editor:
1. Make sure Select tool (‚Üñ) is active
2. Check if object is locked (properties panel)
3. Objects on other floors aren't selectable

#### "Auto-save indicator stuck on 'saving'"
- Usually means slow connection
- Changes are still saved locally
- Try manual save: Ctrl+S

#### "Can't multi-select objects"
Make sure:
1. Select tool (‚Üñ) is active
2. Hold Shift while clicking
3. Or drag selection box on empty space
4. Check that objects aren't locked

## Tips for Great Floor Plans

### Design Best Practices

**1. Start Simple**
- Begin with walls/boundaries
- Add major zones (categories)
- Then add details and labels

**2. Consistent Sizing**
- Similar objects = similar sizes
- Makes map look professional
- Easier for users to understand

**3. Clear Labels**
- Keep names short and clear
- Use common terminology
- Avoid abbreviations unless obvious

**4. Strategic Color Use**
- Group similar types with same colors
- Use contrast for important features
- Don't use too many colors (3-5 main colors)

**5. Add Rich Tags**
- Think like your users
- What would they search for?
- Include common misspellings if needed

**6. Test Your Map**
- Search for common locations
- Try on mobile device
- Ask a friend to test navigation

### Accessibility Considerations

- Use high-contrast colors
- Make text readable (not too small)
- Label all important features
- Test on different devices
- Include accessibility features in tags (e.g., "wheelchair accessible")
